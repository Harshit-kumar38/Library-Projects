# Student Performance Analysis

A Python project that analyzes the performance of 50 students across five subjects using NumPy, Pandas, and Matplotlib.

### Features

* Generate student marks data
* Calculate total marks and percentage
* Assign grades
* Rank students
* Find the topper
* Subject-wise analysis
* Pass/Fail analysis
* Data visualization using charts

### Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Jupyter Notebook

### Subjects

* Maths
* Science
* English
* Hindi
* SST

### How to Run

pip install numpy pandas matplotlib

Open the Jupyter Notebook and run all cells.
